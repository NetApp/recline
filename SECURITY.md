# Security Policy

## Supported Versions

The latest version of the library is always the one that is supported. If problems are found, they would be fixed with a version update.

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |
| older   | :x:                |

## Reporting a Vulnerability

Security vulnerabilities can be privately reported via GitHub: https://github.com/NetApp/recline/security/advisories/new

Issues will be worked on and/or discussed and vulnerabilities that require a change will be addressed in the next release.
